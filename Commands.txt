git add . / <file>  / <folder> - вышрузка изменений на локальный репозиторий
git commit -m <"Text"> - комментирование изменений
git push -u origin <name> - выгрузка изменений на удаленный репозиторий
git pull - загузка обновлений с удаленного репозитория
git branch - просмотр веток
git branch <name> - создание ветки с именем name
git checkout <name> - переход к ветке с именем name
git push --set -upstream origin <name> - выгрузка ветки с именем name на удаленный репозиторий C
